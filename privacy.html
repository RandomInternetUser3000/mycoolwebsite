Privacy Policy

Last updated: December 18, 2025

1. Data collected

This site is primarily static. We may collect minimal data such as form submissions (e.g., contact form) and operational logs (error logs, server access logs). Comments or webhooks, if enabled, include the data you provide when posting or configuring them.

2. Cookies and storage

Local storage is used for preferences (theme, comment drafts) and session cookies are used for admin authentication. These are functional only and not used for ad tracking.

3. Analytics

If enabled, Vercel Web Analytics or similar privacy-friendly analytics may collect aggregated, non-identifying metrics (pages viewed, device type). No cross-site ad tracking is performed.

4. Third-party services

Embedded content and integrations may set their own cookies or collect usage data. Key services and their policies:

  • YouTube (embeds + Data API for latest uploads): Terms, Privacy, API Services Terms
  • Spotify (playlist embed): Terms, Privacy
  • Discord (webhooks / community links): Terms, Privacy
  • Formspree (contact form backend): Terms, Privacy
  • GitHub (admin sign-in and commits): Terms, Privacy
  • Vercel Web Analytics (if enabled): Terms, Privacy

If you sign in to the admin panel, authentication and repository edits use the GitHub API. GitHub may process your account data to complete sign-in and commits. YouTube Data API access is used read-only to surface latest uploads; no sensitive scopes are requested.

5. How data is used

Data is used to operate the site, respond to messages, and improve content. Contact form submissions are used solely to reply to you. Webhook configuration is used to deliver notifications you opt into.

6. Sharing

Data is not sold. It may be shared with service providers strictly to run the site (hosting, analytics) or if required by law.

7. Security

We use HTTPS and limit admin access via authentication and allowlists. No system is perfectly secure—do not submit sensitive personal information through this site.

8. Your choices

You can disable embeds via browser settings, block cookies, or contact us to remove provided content where feasible.

9. Contact

For privacy questions or requests, reach out via contact.html.
