Designing for precision without punishing latency

Every weapon in Aurora Arena is built from a modular ability system. Abilities own their hit windows, stamina taxes, VFX, and camera work, then plug into a shared state machine. That gives me predictable timings while letting me tune each move in isolation.

  • Client-side prediction with server authority keeps hits feeling immediate without desync.
  • Animation events broadcast intent frames ahead so blocks and dodges stay readable.
  • Dynamic slow-mo and bloom respond to successful parries to reward timing mastery.

Telemetry that guides weekly patches

A lightweight analytics pipeline tracks hit accuracy, combo drop rates, and stamina waste per session. I review the dashboards after each private playtest and ship balance patches on Mondays.

  1. Capture granular combat stats per player session.
  2. Tag test cohorts to spot skill-based imbalances.
  3. Automate patch notes from Git commits so testers see what changed.
